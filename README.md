# stepik_tests

Небольшой учебный проект для курса **“Автоматизация тестирования с помощью Selenium и Python”** на Stepik.

Цель проекта — показать, как запускать автотесты Selenium с различными языками интерфейса, передавая язык через параметр командной строки PyTest.


---

## 🛠 Используемые технологии

- Python
- Selenium WebDriver
- PyTest
- Google Chrome
